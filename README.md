## 🔍 What is codesnap?

codesnap helps you create clear, attractive screenshots of your code. With it, you can pick color themes, highlight different programming languages, and save your work as PNG images. It works offline on your Windows computer. codesnap is a free, open-source tool made to give you an easy alternative to other code screenshot apps without any extra cost or clutter.

You do not need to know how to code to use codesnap. If you write or share code, or if you just want neat images for blogs or presentations, codesnap fits your needs.

- Choose from multiple color themes for your code image

- Generate images in PNG format for easy sharing

## ⚙️ Installing codesnap

1. After the download finishes, open your "Downloads" folder or wherever you saved the installer.

3. The Windows User Account Control may ask if you allow the app to make changes. Click "Yes".  
4. The setup wizard will open. Read the instructions on each screen.  
5. Choose "Next" to accept the default installation settings.  
6. The installer will copy the files and install codesnap on your PC.  
7. When the installation finishes, click "Finish" to close the wizard. codesnap is now ready to use.

## 🔓 Opening and Using codesnap

1. Find the codesnap shortcut on your desktop or in the Start Menu under "codesnap".  
2. Double-click to open the program.  
3. On the main screen, you can paste your code or type it in the text box.  
4. Choose the programming language from the dropdown menu to highlight your code properly.  
5. Select a color theme that fits your style or presentation needs.  
6. Preview your screenshot on the right side of the window.  
7. Adjust font size or other options if available.  
8. When you are happy with the look, click the "Export" or "Save as PNG" button.  
9. Choose a folder on your PC where you want to save your image.  
10. Click "Save" and find your screenshot in the selected folder.

## 🌟 Tips for Best Results

- Use smaller chunks of code for clearer images. Long files might be harder to capture in one screenshot.  
- Try different themes to improve readability depending on your background color.  
- Choose the correct language to get the right syntax colors.  
- Save your settings if you find a style you like, so you can use it again with one click.  
- Use PNG images for sharp, high-quality screenshots that work well on websites and documents.
